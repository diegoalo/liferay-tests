Lexicon Skeletons · light & dark previews

Skin

Animated skeleton loading states for the Lexicon component library (Epic LPD-49007), built on the lexicon-vanilla token system: shapes use --color-secondary-l3 on a --color-light-l1 surface, 4px radius, a 1.6s shimmer wave and an opacity fade on repeating rows. Skins follow the kit convention (.light / .dark class), so every skeleton adapts automatically — use Side by side to review both at once.
