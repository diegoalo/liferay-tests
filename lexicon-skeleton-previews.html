Lexicon Components — Skeleton Previews

Animated skeleton loading states for the Lexicon component library, based on the CMS menu skeleton reference (LPD-49007). Shapes use #E7E7ED at 4px radius over a #F7F8F9 surface, with a 1.6s wave (shimmer) animation and a vertical opacity fade on repeating structures.

Shape: #E7E7ED Surface: #F7F8F9 Shimmer: rgba(247,248,249,.6) Radius: 4px Animation: 1.6s infinite Fade: 1 → .85 → .75 → .6 → .4 → .3 → .2

Breadcrumb path segments

Multi Step step indicators + connectors

Form / Input label + field pairs

Not skeleton candidates: Button, Button Group, Radio Button, Badge, Picker, Progress Bar, Pagination, Focus Ring, Slot, Alert. These are either instant-rendering chrome (buttons, badges, pagination), feedback components that appear as a result of an action (alert, progress bar), or utilities (focus ring, slot). Skeletons should only stand in for content that is being fetched.

Reference: Animated Liferay Skeleton (CMS menu) on CodePen · Epic LPD-49007 · Task LPD-49163. Core pattern: shapes in #E7E7ED, 4px radius, shimmer overlay rgba(247,248,249,.6) animated 1.6s infinite, opacity fade on repeating rows.
