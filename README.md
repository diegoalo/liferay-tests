# LIFERAY-TESTS

**HTML PREVIEWS**

A place to drop HTML previews of the work I build in Liferay — fragments, templates, cards, control panel pieces, UI experiments. Raw, in-progress, unpolished on purpose.

---

## WHAT LIVES HERE

Design system pieces and interface tests built for Liferay DXP. Each preview is a snapshot of a piece before it ships.

## HOW IT'S ORGANIZED

- Every piece lives in its own folder.
- Each folder holds the `.html` and its assets — CSS, images, JS when it earns them.

## HOW TO SEE IT

1. Clone the repo.
2. Open the `.html` in your browser.

Or enable GitHub Pages in settings to view previews straight from the web.

## BUILT WITH

HTML · CSS · JAVASCRIPT · LIFERAY FRAGMENTS & TEMPLATES
